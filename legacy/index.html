📌 Обзор проекта Legacy

Legacy — это современный форк Heroku с дополнительными функциями и улучшениями. Является наиболее актуальной версией с последними обновлениями Telegram API.

💡 Примечание: Legacy поддерживает последний слой Telegram с реакциями, видео-стикерами и другими новыми функциями.

✨ Особенности

  • 🆕 Latest Telegram Layer — поддержка новейших функций Telegram
  • 🔒 Enhanced Security — улучшенная система безопасности
  • 🎨 UI/UX — современный интерфейс
  • 📦 Core Modules — улучшенные базовые модули
  • 🔄 Backward Compatibility — полная совместимость с FTG, Hikka, GeekTG модулями

⚙️ Установка

VPS/VDS

# Ubuntu/Debian apt update && apt install git python3 -y git clone https://github.com/Crayz310/Legacy cd Legacy pip install -r requirements.txt python3 -m legacy

Аргументы

  • --proxy-pass — веб-интерфейс через SSH туннель
  • --no-web — консольная установка
  • --root — для root пользователей

📋 Требования

  • Python: 3.9-3.12
  • API: API_ID и API_HASH с my.telegram.org
