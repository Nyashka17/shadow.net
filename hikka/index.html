📌 Обзор проекта Hikka

Hikka — это оригинальный Telegram userbot, разработанный hikariatama. Это основа для многих форков включая Heroku и Legacy.

⚠️ Важно: Hikka архивирован (март 2025). Проект больше не обновляется. Рекомендуется использовать Heroku или Legacy.

📜 История

Hikka была создана как форк Friendly-Telegram (FTG) от Hackintosh5. Затем на её основе были созданы:

  • Heroku — форк от coddrago с улучшениями
  • Legacy — форк от Crayz310, основанный на Heroku

В марте 2025 года проект был архивирован из-за прекращения разработки.

⚙️ Установка

Автоматическая установка (рекомендуется)

# Linux/VPS . <(wget -qO- https://hikariatama.ru/get_hikka)

Ручная установка

# Ubuntu/Debian apt update && apt install git libcairo2 -y git clone https://github.com/hikariatama/Hikka cd Hikka pip install -r requirements.txt python3 -m hikka
💡 Примечание: На VPS добавьте `--proxy-pass` для веб-интерфейса или `--no-web` для консольной установки.

📋 Требования

  • Python: 3.8+
  • Библиотеки: libcairo2
  • API: API_ID и API_HASH с my.telegram.org

📦 Структура модуля

Модули Hikka располагаются в папке hikka/modules/.

# hikka/modules/my_module.py import asyncio from ..import loader, utils @loader.owner class MyModule(loader.Module): """Мой первый модуль""" strings = { "name": "MyModule", "hello": "Привет от модуля!" } async def client_ready(self): """Вызывается при загрузке модуля""" print("Модуль загружен!") @loader.command() async def hello(self, message): await utils.answer(message, self.strings["hello"])

🏷️ Декораторы

Декоратор Описание
@loader.owner Только владелец
@loader.admin Администраторы
@loader.command() Команда с префиксом .
@loader.inline() Inline команда
@loader.loop() Циклическая задача

⚡ Скрытые функции из исходного кода

Скрытые функции из исходного кода Hikka. Эти функции не документированы в официальном вики.

📌 get_prefix() - Получение префикса

module.py
@loader.tds
@loader.owner
class TestModule(loader.Module):
    strings = {"name": "TestModule"}

    async def client_ready(self):
        prefix = self.get_prefix()
        prefixes = self.get_prefixes()

📊 pointer() - Указатели на данные

module.py
@loader.tds
@loader.owner
class DataModule(loader.Module):
    strings = {"name": "DataModule"}

    def __init__(self):
        self.data = self.pointer("key", [])

    async def add_item(self, item):
        self.data.append(item)

    strings = {"name": "ChatModule"}

    @loader.command()
